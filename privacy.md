# Privacy Policy

This is the privacy policy for the **SpliceKit**, macOS Application that is available on GitHub.

**SpliceKit** does not collect any user data. No personal information is transmitted, stored, or used by this application.

**SpliceKit** does not use any third-party analytics or tracking tools. It does not contain any in-app purchases or advertisements.

We use this GitHub Repository for documentation hosting and technical support. You can read GitHub's Privacy Policy [here](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

If you have any questions or concerns, you can [submit an issue](https://github.com/latenitefilms/SpliceKitWebsite/issues), or email us: support@latenitefilms.com
